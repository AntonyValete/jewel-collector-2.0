{\bfseries{Jewel Collector}} é um minigame desenvolvido durante a disciplina de {\bfseries{programação em C\# + .NET do curso Tecnologias Microsoft -\/ Extecamp (Universidade Estadual de Campinas)}}. O objetivo do jogo é coletar jóias para sobreviver em um campo de obstáculos, utilizando o menor número de movimentos possível para conseguir passar para a próxima fase. Para isso, o {\bfseries{Robô (ME)}} coleta as jóias ou recursos e coloca em sua bolsa para ganhar {\bfseries{Energia}}.

A cada fase o mapa do jogo aumenta, e o desafio fica mais difícil, já que cada fase é gerada aleatoriamente e incrementalmente. Os recursos, obstáculos e jóas disponíveis são\+:


\begin{DoxyItemize}
\item {\bfseries{BJ\+:}} Blue Jewel -\/ Score\+: 10 pontos. Restaura 3 pontos de energia;
\item {\bfseries{RJ\+:}} Red Jewel -\/ Score\+: 50 pontos;
\item {\bfseries{GJ\+:}} Green Jewel -\/ Score\+: 100 pontos;
\item \#\#\+: Obstáculo\+: Água;
\item {\bfseries{\$\$\+:}} Obstáculo\+: Árvore. Restaura 3 pontos de energia;
\item {\bfseries{!!\+:}} Obstáculo\+: Radioativo. Elimina 10 pontos de energia;
\end{DoxyItemize}\hypertarget{index_autotoc_md17}{}\doxysubsection{\texorpdfstring{Detalhes de implementação\+:}{Detalhes de implementação:}}\label{index_autotoc_md17}
Os requisitos necessários para a construção do jogo foram passados durante o desenvolvimento da disciplina. A partir deles elaboramos uma \doxysectlink{md__g_1_2e_books_2___extecamp_2jewel-collector-2_80_2checklist}{Checklist}{0} de implementação com as funcionalidades necessárias. A estrutura geral utilizada para sua construção toma como base os detalhes de implementação contidos na checklist. Algumas escolhas também foram tomadas para a otimização do código, bem como funcionalidades que a equipe achou interessante.\hypertarget{index_autotoc_md18}{}\doxysubsection{\texorpdfstring{Compilando Jewel Collector}{Compilando Jewel Collector}}\label{index_autotoc_md18}
Jewel Collector está disponível para Windows, Linux ou Mac\+OS. Para compilar o programa, primeiro é necessário ter feito o download e instalado o \href{https://dotnet.microsoft.com/en-us/download/dotnet/7.0}{\texttt{ .NET 7.\+0}} localmente. Com isso, basta baixar o programa e executá-\/lo utilizando o {\ttfamily dotnet run}.
